# Rugier — Designs du configurateur

Ce dépôt contient les designs de tapis affichés dans le configurateur en ligne Rugier.

Pour ajouter un design : voir **`rugs/_GUIDE.md`**.

Structure :
```
rugs/
  index.json          ← catalogue de tous les designs
  _GUIDE.md           ← mode d'emploi
  cerdan/
    config.json
    module.svg        ← Optical Palazzo (modulaire)
    preview.svg       ← vignette générée
  agata/
    config.json
    preview.svg       ← tapis classique
  ...
```
